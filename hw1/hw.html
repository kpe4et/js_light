как я понял прочитанное по defer/async

defer используется для ускорения загрузки страницы, если скрипт не содержит чего-то сверхнеобходимого, при этом соблюдается объявленная очередность

async используется для игнорирования скриптов при загрузке страницы. Могут загрузиться хоть через полгода, это не помешает использовать страницу. При этом запуск скриптов происходит по очередности их загрузки
